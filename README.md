# 🌤️ Weather_Website

Eine einfache Wetter-Website gebaut mit vanilla JavaScript im MVC-Pattern.
Schriftarten inspiriert von Persona 3 Reload.

## Tech Stack
- HTML / CSS / JavaScript (MVC)
- Bootstrap 5
- OpenWeatherMap API

## Setup
1. Kostenlosen API-Key auf [openweathermap.org](https://openweathermap.org) erstellen
2. `config.example.js` kopieren und in `config.js` umbenennen
3. Eigenen API-Key in `config.js` eintragen
4. Mit Live Server starten

## Features
- Wettersuche nach Stadtname
- Zuletzt gesuchte Städte (max. 3)
- Mobil-kompatibel
